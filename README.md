# MediTrack

**MediTrack** is an Android application for tracking medication intake.  
The project is currently under development.

## Planned Features

- Google account login  
- Adding new medications and dosages  
- Creating daily medication schedules  
- Setting reminders for medication intake  
- Tracking statistics and medication history  
- Other useful features for managing medications

## Project Status

⚠️ Under development. Features are being added gradually.
